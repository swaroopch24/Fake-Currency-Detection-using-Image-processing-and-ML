# 💰 Fake Currency Detection using Machine Learning & Image Processing

## 📌 Description
This project focuses on detecting fake currency notes using **Machine Learning** and **Image Processing techniques**. The system analyzes images of currency notes and identifies whether they are real or fake based on extracted features.

🚧 **Note:** This is a **demo version** of the project. We are still working on improving the model accuracy, adding new features, and enhancing the system.

---

## 🎯 Objectives
- Detect counterfeit (fake) currency notes
- Use image processing for feature extraction
- Apply machine learning for classification
- Provide a simple and efficient detection system

---

## ⚙️ Features
- Upload currency note image
- Image preprocessing (grayscale, resizing, filtering)
- Feature extraction from images
- ML model for prediction (Real / Fake)
- Simple and user-friendly interface

---

## 🧠 Technologies Used
- Python 🐍
- OpenCV (Image Processing)
- NumPy
- Scikit-learn (Machine Learning)
- (Optional) Flask / Streamlit for UI

---

## 🚀 How It Works
1. User uploads an image of a currency note
2. Image is preprocessed using OpenCV
3. Important features are extracted
4. Machine Learning model analyzes the features
5. Output is displayed as:
   - ✅ Real Currency
   - ❌ Fake Currency
